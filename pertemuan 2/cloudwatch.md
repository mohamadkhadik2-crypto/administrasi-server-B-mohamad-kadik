Open the CloudWatch console and select "Alarms" to create a new monitoring metric for our instance![alt text](image-4.png)
Select the specific metric, such as CPU Utilization, and choose the EC2 instance we just created![alt text](image-5.png)
Configure the threshold conditions, for example, setting an alarm if CPU usage exceeds 80%![alt text](image-6.png)
Review the configuration and click "Create Alarm" to finalize the monitoring setup![alt text](image-7.png)
